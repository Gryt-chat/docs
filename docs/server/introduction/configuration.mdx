# Configuration

## Config file

The planet can be configured through the file **gryt.config.js**:

```js title="gryt.config.js"
{
  module.exports = {
    // General information about the server
    title: "Dagobah",
    description: "One of the purest places in the galaxy within the Force"
    tags: ["Friendly", "Games", "DnD", "Bring a friend", "Star Wars"],
    url: "https://gryt-docs.vercel.app/",

    // Server limitations
    video_max_bitrate: 6000,
    voice_max_bitrate: 320,
    file_max_size: "100M",
  };
}
```

### Parameter explanation

| Name          | Default     | Description                                                                                                            |
| ------------- | ----------- | ---------------------------------------------------------------------------------------------------------------------- |
| `title`       | `Dagobah`   | The name of the planet. This is displayed to users in the client.                                                      |
| `--host`      | `localhost` | Specify a host to use. For example, if you want your server to be accessible externally, you can use `--host 0.0.0.0`. |
| `--no-open`   | `false`     | Do not open automatically the page in the browser.                                                                     |
| `--config`    | `undefined` | Path to Docusaurus config file, default to `[siteDir]/docusaurus.config.js`                                            |
| `--no-minify` | `false`     | Build website without minimizing JS/CSS bundles.                                                                       |

:::important

Please note that some functionality (for example, anchor links) will not work in development. The functionality will work as expected in production.

:::
